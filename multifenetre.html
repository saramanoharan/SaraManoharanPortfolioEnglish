        Multifenêtre

        J'ai créé ce site web au cours de ma première année de BTS SIO option SLAM.
        L'objectif de ce projet était d'avoir deux fenêtres distinctes. Dans l'une, s'affiche une liste de liens, et dans l'autre, s'affiche le retour ou les informations associées aux liens sélectionnés. Cela suggère une conception interactive où les utilisateurs peuvent naviguer entre les liens et voir les résultats correspondants dans une fenêtre séparée.
